Početna Pregled slika Slikavanje
  1. nativ API (kamera) - napravljeno
  2. biti installable - napravljeno
    • web: pojavi se ikonica u url bar-u za instalirati aplikaciju
    • mobile: iskoci gumb za instalaciju na Home screen
  3. imati barem jednu cacheing strategiju - napravljeno
    • slike: implementirana "Stale-while-revalidate" strategija
    • ostale datoteke: "Cache first, falling back to network" strategija
  4. offline rad barem za app shell - napravljeno
    • provjera kao na predavanju (odabrati offline mode u application tabu u inspect elements u Chromu)
  5. background sync - nije napravljeno
  6. push notifikacija (npr. nakon što prođe bg sync možete poslati push notif) - nije napravljeno
  7. demonstrirati barem na jednom primjeru progressive enhancement/gracefull degradation
    • jedno od najznačajnijih poboljšanja je instalabilnost - čime se (posebice na mobilnom uređaju) poboljšava korisnikovo korištenje - web aplikacija se "maskira" u nativnu - prednost dodavanja na Home screen
    • "re-engageable" pomoću push notifikacija
    • mogućnost rada u offline načinu
